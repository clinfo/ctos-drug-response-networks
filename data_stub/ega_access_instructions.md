# Controlled-access sequencing data

The raw whole-exome and RNA-seq datasets used by this project are not redistributed in this repository.

## Registry accession

- Dataset accession: `EGAD50000002401`
- Registry record: `https://ega-archive.org/datasets/EGAD50000002401`

## Access model

This dataset is handled through controlled access. Use the registry landing page to inspect the current access route and release state.

## What Route B expects

After access is granted, stage the raw files outside the repository, run the optional raw-data helpers only as needed, and convert the outputs into the Route A asset schema.
