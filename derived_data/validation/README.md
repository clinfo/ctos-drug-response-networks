This directory contains the bundled validation assets used by the public Route A rebuild.

- `primer_list.tsv` and `primer_list.xlsx` reproduce the released qPCR primer list.
- `qpcr_long.tsv` is an exact long-format reconstruction of the archived qPCR support tables.
- `score_model_*_coefficients.tsv` are the released logistic-coefficient tables.
- `score_summary.tsv` is a release-safe recomputation of per-patient secondary score summaries from the bundled qPCR matrix and released coefficients.
