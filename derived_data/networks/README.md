This directory contains the bundled network assets used by the public Route A rebuild.

- `common_network_edges.tsv` and `resistant_missing_edges.tsv` were reconstructed from exact archived notebook edge sets.
- `patient_network_edges.tsv` and `patient_network_nodes.tsv` are curated public Route A panel-support subsets ordered to regenerate the LDN193189 schematic patient-network panels.

See `docs/panel_support_asset_manifest.tsv` for exact provenance and fidelity classes.
