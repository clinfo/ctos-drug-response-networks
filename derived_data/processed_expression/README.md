This directory contains the bundled expression-support assets used by the public Route A rebuild.

- `pathway_logfc_long.tsv` is an exact archived support-panel extraction averaged across archived triplicates.
- `pathway_expression_long.tsv` is a release-safe pretreatment support matrix for the same public Route A panel universe; it is not claimed as a full exact upstream pretreatment export.
- `deg_shared.tsv` contains the exact 36-gene shared LDN193189 DEG panel reconstructed from publication-support values.

These files are release-safe panel-support assets rather than full upstream RNA-seq exports.
