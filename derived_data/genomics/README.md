This directory contains the bundled genomics assets used by the public Route A rebuild.

- `tmb.tsv` is an exact release-safe summary value table.
- `pathway_mutations.tsv` and `pathway_cna.tsv` are curated pathway-level comparator summaries aligned to the manuscript-described examples.

These files are sufficient for the public comparator-matrix rebuild, but they are not a full released cohort-wide MAF or CNA export.
